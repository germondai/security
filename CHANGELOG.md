# Changelog

All notable changes to **Germond's Security** are documented here.
The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html) once it ships `1.0`.

> 🧪 The project is currently **pre-1.0** (`0.0.0` across every workspace).
> While in `0.x`, anything may change. Pin with care.

## Pre-release history

The repository was developed privately before this public release. See
[`git log`](https://github.com/germondai/security/commits/main) for the full
pre-release history — it includes the initial scaffolding of the core crypto
library (`packages/security`), the CLI (`apps/cli`), and the Vue 3 web app
(`apps/web`).

## Release types (planned)

For each future release we use the following subsection headings, in order:

- **Added** — new features.
- **Changed** — changes in existing functionality.
- **Deprecated** — soon-to-be removed features.
- **Removed** — now-removed features.
- **Fixed** — bug fixes.
- **Security** — vulnerability fixes (also published via GitHub Security Advisories).
